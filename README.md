# AI_Exercise_Generator
This generator is a fully automated Python pipeline that creates complete workout videos from scratch. Given a set of exercises, it downloads video clips from Pexels (or uses your own stock footage), generates AI voiceover narration, mixes background music, renders a countdown timer overlay, and exports a finished 1920×1080 MP4 


Key highlights:
•	Fully automated end-to-end: from exercise list to finished video
•	Smart Pexels search with bodyweight/home-only keyword filters
•	AI voiceover via Google TTS with exercise-specific coaching cues
•	Countdown timer overlay with colour-changing progress bar
•	5 randomised intro card styles per run for variety
•	Background music management with no-repeat tracking
•	YouTube and TikTok upload support included
